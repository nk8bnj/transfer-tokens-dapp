# Try out dApp

[Demo](https://nk8bnj.github.io/transfer-tokens-dapp/) to open the decentralized application.

## Getting Started

```
$ git clone https://github.com/8bitnikita/transfer-tokens-dapp.git
```

### Run the dApp
```
$ yarn install
$ yarn start
```

#### Features

- React + TypeScript
- Hardhat + Ethers
